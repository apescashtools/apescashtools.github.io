Gambling Apes Selector

The unofficial but useful tools for the gambling.apes.cash NFT collection on SmartBCH

Instructions: Search for the combination of traits that you like by checking some options and then hit the search button.

There is a small fee in sBCH for each search.

Click on a NFT's name to be taken to the Oasis NFT marketplace. After the search has finished, the site automatically checks if the NFTs are offered for sale. This process may take a while.

Terms of Service: Only send payments on the SmartBCH network. No responsiblity for lost funds. Use at your own risk. This website is in the BETA stage. No user data is collected.

To use multiple selections on desktop, click while pressing Ctrl (Cmd on Mac).
